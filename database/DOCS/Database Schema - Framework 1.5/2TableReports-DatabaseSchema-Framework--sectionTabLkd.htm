FLAG_DEFINE_USE_PROFILE_LEVEL int 
Indicator to define the usability of the profile. These are the valid scenarios:

I - LISTING USERS
* When the flag = 0, the logged user can edit/delete other users with the
PROFILE_LEVEL > logged user's Profile Level
* When the flag = 1, the logged user can edit/delete other users with the
PROFILE_LEVEL > logged user's profile Level and other users with the same
logged user's Profile

II - EDITING/CREATING USER
* When the logged user is editing/creating a new users and the flag = 0, the
available Profiles has to be Profiles with value PROFILE_LEVEL > logged
user's Profile Level
* When the logged user is editing/creating a new users and the flag = 1, the
available Profiles has to be Profiles with value PROFILE_LEVEL > logged
user's Profile Level and the logged user's Profile

III - LISTING PROFILES
* When the flag = 0, the logged user can edit/delete other profiles with the
PROFILE_LEVEL > logged user's Profile Level
* When the flag = 1, the logged user can edit/delete other profiles with the
PROFILE_LEVEL > logged user's profile Level and its own profile.

IV - EDITING PROFILES
* When the logged user is editing a profile and the flag = 0, there isn't a
check box to select the value of the flag and this one has as a default
value zero (0) and the combo with avaliable Profile Levels has to be
PROFILE_LEVEL > logged user's Profile Level
* When the logged user is editing a profile and the flag = 1, the check box
to select the value of the flag is gotten from the database and the combo
with avaliable Profile Levels has to be PROFILE_LEVEL > logged user's
Profile Level. Only when the user is editing his own profile, the combo with
avaliable Profile Levels has to be PROFILE_LEVEL >= logged user's Profile
Level.

V - ADDING PROFILES
* When the logged user is adding a profile and the flag = 0, there isn't a
check box to select the value of the flag and this one has as a default
value zero (0) and the combo with available Profiles Levels has to be
PROFILE_LEVEL > logged user's Profile Level
* When the logged user is adding a profile and the flag = 1, there is a
check box to select the value of the flag and this one has as a default
value zero (0) and the combo with available Profiles Levels has to be
PROFILE_LEVEL > logged user's Profile Level
